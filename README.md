# Sentiment-Analysis
Brief Introduction: It encapsulates the code for conducting sentiment analysis on posts obtained from x by politically elected candidates.

# Code structure
Imports.py
Centralised imports and shared helper utilities used across the scripts.

Dataset Creation.py
Converts the raw Excel workbook into a long format dataset with one row per post.

Wordcloud.py
Generates word clouds by state and, optionally, by candidate.

Frequency and Zipf Plots.py
Generates frequency plots, candidate polar bar charts, candidate histograms, and Zipf distribution outputs.
